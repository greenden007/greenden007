## Some Recent Completed Work

### [LoL Draft Agent](https://github.com/talkeridan17/TrynDraft)
An agentic AI system that assists with League of Legends champion drafting. Uses a **RAG pipeline backed by an LLM** to provide human-readable explanations for draft decisions — not just recommendations, but the reasoning behind them.

- LLM-powered explainability via retrieval-augmented generation
- Combines game state context with strategic knowledge retrieval

### [What-If](https://github.com/greenden007/what-if)
A two-stage NLP pipeline that **classifies and rewrites dialogue and behavior** in movie/TV scripts to modify how characters are portrayed.

- Stage 1: Classifier identifies character-specific dialogue and behavioral cues
- Stage 2: LLM-based rewriter modifies tone, personality, and speech patterns
- Useful for adaptation, accessibility, and creative exploration
